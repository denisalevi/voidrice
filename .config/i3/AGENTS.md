This folder contains my i3 configuration.

`config` is the main i3 config.
`i3blocks.conf` configures the i3blocks status bar.
`keybindings` contains keybinding definitions included by `config`.
`scripts/` contains i3blocks helper scripts (battery, volume, cpu, etc).

I use i3blocks as status bar, please read the docs here if suggesting anything about i3blocks: https://vivien.github.io/i3blocks/

My dotfiles use a custom git wrapper command called `dotfiles` (use `dotfiles status .` in this folder). Tell me what changed; I will handle committing unless I explicitly ask you to do it.
